# 필기: 이론·객관식

빅분기 필기는 4과목 80문항 4지선다.

## 출제 범위 (과목별)

| 과목 | 비중 | 주요 주제 |
|------|------|----------|
| 1. 빅데이터 분석 기획 | 20% | 데이터 처리 기술, 분석 방안 수립, 거버넌스 |
| 2. 빅데이터 탐색 | 20% | EDA, 결측치/이상치, 변수 선택, 차원 축소 |
| 3. 빅데이터 모델링 | 30% | 회귀/분류/군집, 앙상블, 평가 지표 |
| 4. 빅데이터 결과 해석 | 30% | 평가, 시각화, 분석 결과 활용 |

## 작성 원칙

- **함정 선지 1~2개**: 헷갈리기 쉬운 유사 개념을 오답으로 (예: bagging vs boosting)
- **명확한 정답 1개**: 절대 두 개의 정답이 동시에 가능해선 안 됨
- **부정형 주의**: "다음 중 옳지 않은 것은?" 같은 부정형 문제는 별도 태깅
- **계산 문제**: 직접 계산이 필요한 경우 단계별 풀이를 explanation에 명시
